Web Harvester for Sitecore

Thank you for using this extension!

This Extension enables content administrators/developers to harvest any website's content into Sitecore

To migrate a website's content into Sitecore, you must right-click on the intended website page and select 'Inspect Element' or press 'F12' to launch Developer Tools. Navigate to 'Web Harvester' panel in the Developer Tools to begin import process by selecting DOMs and Sitecore Item Path, Template & Fields.

IMPORTANT: For importing media, you will need to install the below Sitecore Powershell Module in your Sitecore instance and specify Sitecore credentials(ensure appropriate create permissions for media library) in the 'Settings' tab of 'Web Harvester' DevTools Panel. This Chrome Extension does not collect any information including credentials from you.
