Web Harvester for Sitecore

Thank you for using this extension!

This Extension enables content administrators/developers to harvest and import any website's content into Sitecore

PREREQUISITES:

  • Ensure that either ServicesOnPolicy or ServicesLocalOnlyPolicy(if Sitecore is hosted locally) is configured for Sitecore.Services.SecurityPolicy in the Sitecore.Services.Client.config of your Sitecore instance
  • For importing media into Sitecore,
    • Ensure Sitecore Powershell Extensions is installed in your Sitecore instance
    • Install the below Sitecore Powershell Module in your Sitecore instance, https://github.com/SubbuRamanathan/web-harvester-for-sitecore/releases/tag/v1.0
    • Enable 'restfulv2' in Sitecore Powershell Config as described here
    • Specify Sitecore credentials(ensure appropriate create permissions for media library) in the 'Settings' tab of 'Web Harvester' DevTools Panel. This Chrome Extension does not collect any information including credentials from you. Please review the privacy policy for more details. Alternatively, anonymous access can be enabled for 'restfulv2'(not recommended).

To migrate a website's content into Sitecore, you must right-click on the intended website page and select 'Inspect Element' or press 'F12' to launch Developer Tools. Navigate to 'Web Harvester' panel in the Developer Tools to begin import process by selecting DOMs and Sitecore Item Path, Template & Fields.
